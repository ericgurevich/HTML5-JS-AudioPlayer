Audio Player Demo

Eric Gurevich

Instructions: include fw.js in your script src. To load an audio player into a "content" div, call load(source, art, artist, track, content).

  • source: file url of audio file
  • art: file url of album art image
  • artist: artist name
  • track: track name
  • content: div where you want the player to load
load('audio/Autumn Leaves.mp3',
'https://upload.wikimedia.org/wikipedia/en/8/82/Bill_Evans_Trio_Portraits_in_Jazz.jpg',
'Bill Evans',
'Autumn Leaves', 
'content')
                    
load('audio/19 Through The Wire.mp3',
'https://upload.wikimedia.org/wikipedia/en/a/a3/Kanyewest_collegedropout.jpg',
'Kanye West',
'Through The Wire', 
'content')
